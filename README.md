#  Eat Good, Save Food

Вебзастосунок для зменшення харчових відходів шляхом продажу залишків страв зі знижкою.

## Стек технологій
- **Backend:** Python 3, Flask, Werkzeug, PyJWT
- **База даних:** SQLite (вбудована, файл `eat_good.db`)
- **Frontend:** HTML5 / CSS3 / JavaScript (без фреймворків)

## Швидкий старт

### 1. Встановити залежності
```bash
pip install Flask Werkzeug PyJWT python-dotenv
```

### 2. Запустити
```bash
cd eat_good_save_food
python app.py
```

Відкрийте: **http://localhost:5000**

База даних та демо-дані створяться автоматично при першому запуску.
